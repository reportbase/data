        Steps To Get <br>Familiar With RepbaView.<p> 
        <div align=left>
        <div style="font-size:medium">
            1. Interact with the top iFrame image. Click the top iFrame image to stop panoramic motion and click 
            the right/left side of image to move forward/backward thru the images.<p><p>          
            2. Select imagegroup subgroups using the image menu. After 
            <a href=/?p=boss/ultra&a=1&k=0004&s=1&f=1 target="_parent">click to start </a>the Ultra image group, view various 
            image galleries by clicking the left icon image menu subgroup buttons. Click the browser back arrow to return 
            to this page. <p><p>
            3. Perform actions defined by the <a target=ifr
            href="/?p=boss/ultra&k=0001&a=1&s=2&f=1">RepaView Images</a> page.<p><p>
        </div><p>

        <div style="font-size:medium;">
            4. Enable panorama 
            and slideshow motion using the command menu with various image subgroups. After motion type is 
            selected, click the command menu icon to clear the displayed buttons. Click anywhere on the screen 
            to stop motion and click the toolbar right-arrow to start motion. Refer to the command menu help button for 
            command information. There are two action sequences. 1)                                    
            <a href=/?p=boss/king&a=1&k=0003&s=1&f=1 target="_parent"> Click to
            start </a> the Giraffes subgroup of the animal King imagegroup to perform the actions. 
            Click the browser back arrow to return to this page. 2) Tap the GIF image below to start the Ocean subgroup 
            of the poke imagegroup and perform the above actions. 
            Click the browser back arrow to return to this page.<p>
                </div><br>

            6. View imagegroup of various framed slideshows. The image subgroups have different numbers of images. Select a
            few of the fun slideshows enabled by the top left image menu icon
