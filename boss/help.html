FSPIV  “FizPiv” Application Command Summary

Full Screen Panoramic Image Viewer

Menus

Upper Left Icon: Image group menu. Selects Image group to display.

Upper Right Icon: Home menu. Selects Index thumbnail view, panoramic view, filmstrip view, next image, or zoom options.

User Interaction

  • To stop motion display, click the home menu icon to stop motion and enable the home menu.

  • When full image motion is stopped, pan left/right to move left/right  of current image and pan up/down to move to the previous/next montage image.

  • The home menu zoom button enables screen magnification where 1/3rd of the full image height is displayed as selected by the Next button. Use the left/right arrow to move horizontally across a magnified image.

  • When zoom is disabled, the Next button select the next or previous image.

  • The index thumbnail display is helpful to quickly scan images or find a specific image.

  • During all motion displays, a left/right swipe change the motion direction. For  left/right cursor panning motion, the left side of screen has slower motion and right side of screen has faster motion. For up/down motion, the upper screen has slower motion and lower screen has faster motion. Scrolling speeds graduate between screen end points.

URL Attributes 

The FSPIV url string specifies an image sub-group to display or initiates a menu enabling users to manually select an image sub-group to display. An image sub-group consists of images with similar orientations and aspect ratios. Users can create bookmarks to sub-groups or specific images. A user can directly link to any image by copying the url from the browser. Url selections are functional for the 1st menu group. A summary of the key URL attributes are: p = Group (required); k = Sub-Group (default - first); e = Image (default - first); d = Display (default – 0), a= motion or slideshow options: a=0; No motion, a=1; Enables cylindrical motion, a=>40 Enables filmstrip motion in ms. (a=1200 has image delay of 1.2sec), a= not specified, default a=1500 (filmstrip image motion delay of 1.5sec).
