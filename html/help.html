FSPIV  “FizPiv” Application Command Summary

Full Screen Panoramic Image Viewer

User Commands for cellphone, tablet, or PC

  • Upper Left Icon: Image group menu. Selects Image group to display.

  • Upper Right Icon: Command menu. Selects Index thumbnail view, panoramic view, filmstrip view, next image, or zoom options.

  • To stop a motion (parametric, filmstrip, index) display, click the upper right command menu icon to stop motion and enable the command menu.

  • When  full image motion is stopped, pan left/right to move left/right  of current image and  pan up/down to move to the previous/next montage image. Click the upper left image sub-group menu to select images to display. Click the command menu options to start slideshow motions or select the next or previous image. Click left of Next for previous.

  • The command menu zoom button enables screen magnification where 1/3rd of the full image height is displayed as selected by the Next button. Use the left/right arrow to move horizontally across a magnified image.

  • When Zoom is disabled, the Next button selects the next or previous image.

  • The index thumbnail display is helpful to quickly scan images to find a specific image.

  • During all motion displays, a left/right swipe change the motion direction. For up/down cursor panning motion, the left side of the screen has slower motion and the right side of screen has faster motion. For left/right motion, the upper screen has slower motion and lower screen has faster motion. Scrolling speeds graduate between screen end points.

URL Attributes

The FSPIV url string specifies an image sub-group to display or initiate a menu to enable users to manually select an image sub-group to display. An image sub-group consists of images with similar orientations and aspect ratios. Users can create bookmarks to sub-groups or specific images. A user can directly link to any image by copying the url from the browser for that image.

The FSPIV url string specifies an image sub-group to display or initiates a menu enabling users to manually select an image sub-group to display. An image sub-group consists of images with similar orientations and aspect ratios. Users can create bookmarks to sub-groups or specific images. A user can directly link to any image by copying the url from the browser. Url selections are functional for the first menu group. Subsequent image sub-group utilize init.ini configuration parameters.  A summary of the key URL attributes are: p = Group (required); k = Sub-Group (default - first); e = Image (default - first); d = Display (default – 0), a= motion or slideshow options: a=0; No motion, a=1; Enables cylindrical motion, a=2; Enables the index thumbnail display, a=>40; Enables filmstrip motion in ms. (a=1200 has image delay of 1.2sec), a= not specified, default a=1500 (filmstrip image motion delay of 1.5sec).



Example URL start-up options. 

https://repba.com/?p=boss/interiors - Starts interior sub-group with display of sub-group menu for user selection of sub-group.

https://repba.com/?p=boss/interiors&k=0000&a=2000 - Starts interior sub-group 0000 with filmstrip motion of 2.0 seconds.

https://repba.com/?p=boss/interiors&k=0001&a=1 Starts interior sub-group 0001 with parametric motion.
