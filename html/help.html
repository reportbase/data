FSPIV Help

Using a cellphone, tablet, or PC, the following is a summary of FSPIV user command menu options and interaction.. 

  • Upper Left Icon: Image group menu. Selects Image sub-group to display.

  • Upper Right Icon: Command menu. Selects full screen (PC & Android only) view, panoramic view, filmstrip view, next image, zoom, help options.

  • To stop motion (parametric, filmstrip, thumbnail) display, click anywhere on the screen.

  • When full image motion is stopped, pan left/right or up/down to move to the next/previous image. Also select the bottom screen “3 dots” icon to enable the thumbnail display.

  • The command menu zoom button enables screen magnification where 1/3rd of the full image height is displayed as selected by the next button. If filmstrip is enabled, the view will automatically move across each of the 3 regions. 

  • The next button selects the next or previous image.

  • The full screen button enables the full screen image display for PC and Android devices.  Exit full screen by selecting the Esc or display “X” buttons. Full screen must be disabled to select the FSPIV menus. 

  • Press the right side of the screen to move images right to left. Press the left side of the screen to move images left to right.  A pressed screen moves the image slowly initially and then accelerates. 

  • During motion displays, a left/right swipe changes the motion direction. For up/down cursor panning motion, the left side of the screen provides slower motion than the right side of screen. For left/right motion, the upper screen has slower motion than the lower screen. Scrolling speeds graduate between the screen end points.

A summary of the key URL attributes are: p = image group (required); k = sub-group (default - first); e = Image (default - first); a = slideshow: a=0; no slideshow, a=1; cylindrical motion, a=2 filmstrip, a=3 burst mode digital images.
